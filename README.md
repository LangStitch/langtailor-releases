# LangTailor — Downloads

Public download host for the **LangTailor** desktop IDE (the downloadable
LangGraph IDE with the LangStitch visual canvas built in).

The LangTailor source code is private. **Release builds are published here** so
they stay publicly downloadable:

- **Portable Windows / macOS builds** — see [Releases](https://github.com/LangStitch/langtailor-releases/releases)
- **VS Code / VSX extension** — install **LangTailor Canvas** from
  [Open VSX](https://open-vsx.org/extension/langstitch/langtailor-canvas)

Download page: **[langtailor.langstitch.com](https://langtailor.langstitch.com/)**

> Releases are produced by the private `LangStitch/langtailor` build pipeline and
> pushed to this repository's Releases.
